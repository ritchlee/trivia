# Overview

The best music trivia app for mobile!

# Local Environment Setup

## Prerequisites

- Postgres 17.0
- Ruby 3.3.34 / Rails 8.0.2
- Node 22.4.1
- XCode / IOS Emulator
- Android Studio / Andoird Emulator

## Server

- `cd backend`
- `bundle install`
- `bin/rails s`

## Mobile

- `cd mobile`
- `yarn start`
- `yarn ios` (for iOS)
- `yarn android` (for Android)
